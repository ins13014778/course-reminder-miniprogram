## 7. 风险关注点

1. 当前后台连接真实库，删除类操作需谨慎操作。
2. `courses` 表真实结构与本地旧实体存在差异，新增功能前必须继续以真实表结构为准。
3. 真实数据会变化，测试结论优先看“接口成功、字段正确、页面一致”，不要死盯某个固定数值。
4. 如果后端重新改环境配置，需重新确认 `.env` 是否仍指向 `dawdawd15-8g023nsw8cb3f68a`。
