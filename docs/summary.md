## 🚀 快速启动

### 后端本地测试
```bash
cd backend
npm install
npm run start:dev
```

### 小程序开发
1. 使用微信开发者工具打开 miniprogram 目录
2. 配置 AppID
3. 修改 utils/request.ts 中的 API 地址

## 📝 技术亮点

1. **模块化设计**：OCR、解析器、消息发送均独立抽象
2. **可扩展架构**：预留接口方便替换真实服务
3. **TypeScript 全栈**：类型安全
4. **CloudBase 集成**：数据库、云托管一体化
